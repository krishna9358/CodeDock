# Codedock

1. Backend
2. Frontend

Backend : nodejs
Frontend : React

## Backend

-   /template : to get the template based on the user prompt project type
-   /chat : to get the chat response from the LLM to generate the actual ui code according to the user prompt


## Frontend

-   / : to render the home page
-   /code : to render the generated ui code


## LLM

-   GROQ (LLAMA 3.1)

## Running the project

-   cd backend
-   npm install
-   npm run dev

-   cd frontend
-   npm install
-   npm run dev

## Future Scope

-   Add more LLMs
-   Add more templates
-   Add User Space for history of user's projects
-   Add Authentication
-   Add own LLM for more accuracy and control over the generated code
